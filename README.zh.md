## ✨ 主要功能

- **⚡ 无服务器编码**：视频不上传至服务器。处理完全在浏览器内完成。
- **✂️ 精确剪辑**：支持 0.01 秒增量的微调和逐帧导航。
- **🔗 视频合并**：将多个 MP4 文件合并为一个，并支持自定义排序。
- **🔄 音视频同步 (Sync)**：根据外部音频文件的长度自动调整视频帧率 (FPS)。
- **🖼️ 高质量帧捕获**：立即将任何场景提取为 PNG 图像。
- **⌨️ 键盘快捷键支持**：使用方向键 (←, →) 和空格键进行高效的时间轴控制。
- **🎨 现代 UI/UX**：shadcn/ui 风格的极简且精致的设计。
- **🔒 隐私保护**：所有工作均在本地完成，让您可以放心地编辑敏感视频。

## 🛠️ 技术栈

- **引擎**: [FFmpeg.wasm](https://ffmpegwasm.netlify.app/) (WebAssembly)
- **前端**: Vanilla HTML5, CSS3 (Modern Zinc Theme), JavaScript (ES6+)
- **SEO**: Meta Tags, robots.txt, sitemap.xml 优化
- **部署**: GitHub Pages (通过 COI Service Worker 支持 SharedArrayBuffer)

## 📖 快速入门

1. **上传文件**：拖放或点击选择 MP4 文件。
2. **设置区间**：使用底部的滑块或键盘方向键设置起点/终点。
    - `← / →`: 移动 0.05s
    - `Shift + ← / →`: 移动 0.01s (帧单位)
3. **捕获/保存**：
    - `捕获当前帧`：将当前场景保存为 PNG。
    - `修剪并保存视频`：裁剪所选部分并下载为新文件。
