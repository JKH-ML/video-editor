# 🎬 Video Editor

**高速、プライベート、そしてブラウザベースの MP4 ビデオトリマー。**

> **無制限無料 + オープンソース**  
> **会員登録不要 ・ 広告なし ・ ウォーターマークなし ・ データ収集なし**

[English](../README.md) | [한국어](README.ko.md) | [简体中文](README.zh.md) | [日本語](README.ja.md)

`Video Editor` は、サーバーへのアップロードなしでブラウザ上で即座に動作する高性能な MP4 編集ツールです。 **FFmpeg.wasm** を搭載しており、すべてのビデオ処理がデバイス上でローカルに行われるため、データの流出を心配することなく高速かつ安全に編集できます。

## ✨ 主な機能

- **⚡ サーバーレスエンコーディング**: ビデオをサーバーにアップロードしません。処理はすべてブラウザ内で行われます。
- **✂️ 精密なカット編集**: 0.01秒単位の微調整とフレーム単位のナビゲーションに対応。
- **🔗 ビ디오結合**: 複数の MP4 ファイルを任意の順序で1つに結合。
- **🔄 オーディオ同期 (Sync)**: 外部オーディオファイルの長さに合わせて、ビデオのフレームレート (FPS) を自動調整。
- **🖼️ 高品質フレームキャプチャ**: 任意のシーンを PNG 画像として即座に抽出。
- **⌨️ キーボードショートカット対応**: 矢印キー (←, →) とスペースキーによる効率的なタイムライン制御。
- **🎨 モダンな UI/UX**: shadcn/ui スタイルのミニマルで洗練されたデザイン。
- **🔒 プライバシー保護**: すべての作業がローカルで完結するため、機密性の高いビデオも安心して編集できます。

## 🛠️ 技術スタック

- **エンジン**: [FFmpeg.wasm](https://ffmpegwasm.netlify.app/) (WebAssembly)
- **フロントエンド**: Vanilla HTML5, CSS3 (Modern Zinc Theme), JavaScript (ES6+)
- **SEO**: Meta Tags, robots.txt, sitemap.xml 最適化
- **デプロイ**: GitHub Pages (SharedArrayBuffer 対応のため COI Service Worker を使用)

## 📖 クイックスタート

1. **ファイルをアップロード**: MP4 ファイルをドラッグ＆ドロップするか、クリックして選択します。
2. **範囲を設定**: 下部のスライダーまたはキーボードの矢印キーを使用して、開始点と終了点を設定します。
    - `← / →`: 0.05秒移動
    - `Shift + ← / →`: 0.01秒移動 (フレーム単位)
3. **キャプチャ/保存**:
    - `現在のフレームをキャプチャ`: 現在のシーンを PNG として保存します。
    - `ビデオをトリミングして保存`: 選択したセクションをカットし、新しいファイルとしてダウンロードします。
